What BYOJ is not

  • Not part of cross-team rollups (verdicts tagged provenance: "custom").
  • Not eligible for the named panels (diverse-3, frontier-3, cheap-5).
  • Cannot share a dimension name with a built-in judge.
  • Cannot ship custom JSON parsing — the system enforces a deterministic parser.
  • Once a verdict has been emitted under @v{n}, that version is immutable; edits create @v{n+1}.
