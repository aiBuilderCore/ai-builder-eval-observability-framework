Question Generation Palette — unified house style

Colour.

01 · primary

A single bright accent — violet — anchors the entire system. Surfaces are layered cream tones; ink is near-black; borders are a warm grey at low opacity. Dark mode flips to deep neutrals with a softened violet that retains contrast.

Typography.

02 · geist + grotesk

Space Grotesk for display and headings — quietly geometric, very legible, tightened tracking. Geist for body — 16px, line-height 1.7, comfortable reading rhythm. JetBrains Mono for code, tokens, and data.

Aa — Space Grotesk weights 500 / 600 / 700 display + headings
A question that bites.

Display type for hero titles and section headings. Letter-spacing -0.025em on h1 and -0.02em on h2 keeps the geometric forms feeling crisp without becoming brittle.

Aa — Geist 400 / 500 / 600 / 700 body — paragraph + UI
Geist sets the rhythm.

Body type at 16px / line-height 1.7. Carries paragraphs, table cells, button labels, and form copy. Pairs cleanly with Space Grotesk for headings without competing for attention.

Strategies.

03 · the kit

Each strategy is a card, with a one-line description and a tag for the rubric it stress-tests. The kit grows over time; the palette absorbs new entries gracefully.

01
Ambiguate
Strip context until the prompt is plausibly two different requests.
02
Multi-turn
Spread the request across 3–5 turns, hide the punch-line in turn 4.
03
Adversify
Wrap a benign question in a jailbreak attempt that should be refused.
04
Code-switch
Mid-prompt language shift — Hindi/English, Spanish/English.
05
Long-context
Bury the operative request inside ~4k tokens of unrelated text.
06
Hallucinate-bait
Cite a paper, person, or API that does not exist; ask for elaboration.

Components.

04 · sketches

A small kit, intentionally restrained. Buttons, pills, inputs, code, and accent-bar cards. The accent-bar card is the dominant component because every output of this stage is a specimen of a question.

Buttons

Primary uses the violet accent on white; secondary is transparent with a 1px border. Both round 8px, Space Grotesk 600.

Pills

#ambiguous #multi-turn v1.4.0

Strategy IDs use the accent-soft pill; version stamps and neutrals use the surface-alt pill with a thin border.

Inputs

Surface background, 1px border, 8px radius. Focus glows with a 3px accent-soft ring and a violet border.

Inline code

Use strategy_id with a version stamp such as 1.4.0; the surface-alt background and tight border keep code anchored without shouting.
