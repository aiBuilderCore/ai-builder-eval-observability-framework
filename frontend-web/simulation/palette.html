Simulation Palette — unified house style

The same Verity tokens applied to the Simulation surface. Violet on warm cream — Space Grotesk for headlines, Geist for tables of runs, JetBrains Mono for the trace stream. Light is default; dark switches automatically on system preference.

Typography.

02 · display · body · mono

Space Grotesk for the headlines that announce a run; Geist for the operational body — adapter names, target IDs, run-set tables; JetBrains Mono for the trace stream that scrolls beneath every run in progress.

Aa — Space Grotesk500 / 600 / 700display
Run #4912, complete.

A run is the act of taking a generated question-set, pushing it through the target adapter, capturing every turn, and writing the resulting RunTrace to the simulation index. Every run has an ID; every ID is a citation a judge can hang a verdict on.

Aa — Geist400 / 500 / 600 / 700body — tables, adapters
Operational, by default.

Geist carries the operational language of this product: adapter manifests, target IDs, latency tables, retry metadata. It's clear and characterful without becoming decorative — exactly the register a runner needs when the user is staring at six concurrent runs.
