Self Heal

Self Heal turns a flagged problem into a fix. When a quality gate or monitor trips, Verity opens an incident, works out the likely cause, rehearses a candidate fix against your test users, and then either ships it automatically or sends it for human review, based on the rules you set.

The closed loop

gate → RCA → simulate → remediate, governed by policy
01detect

Gate Engine

P0 / P1 / P2 threshold logic. A breached gate or drift monitor opens an incident — committed as versioned config.

02diagnose

RCA Agent

LLM-driven span-tree analysis attributes the failure to a cause class and the exact span path that produced it.

03rehearse

Simulator

Rehearses the candidate fix against a persona-driven practice world before anything ships.

04apply

Remediation Registry

A fixed vocabulary of safe actions — prompt rewrite, re-rank tune, guardrail tweak, fall-back, KB update, circuit break.

05govern

Policy DSL

Confidence bands decide ship-vs-escalate: high-confidence fixes auto-apply in-band, the rest open a ticket out-of-band.
